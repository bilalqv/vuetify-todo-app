# vuetify-todo-app


## Project setup

```
git clone https://github.com/crediblebilal/vuetify-todo-app.git
```

```
cd vuetify-todo-app
```

```
npm install
```

```
npm run serve
```



This app fetches data from the server created by GaganDeep, so the functionalities of the app may not be working right now.
